# 관리자 신고 처리 가이드

## AI 분석의 역할

신고 AI는 신고를 최종 처리하지 않는다.
AI 분석은 신고 원문, 대상 게시글, 피신고자의 누적 신고 맥락을 요약하고 관리자 조치 방향을 제안하는 참고 정보다.

## 위험도 기준

LOW는 신고 상세가 부족하거나 정책 위반 근거가 약한 경우다.
MEDIUM은 정책 위반 가능성이 있으나 추가 확인이 필요한 경우다.
HIGH는 반복 신고, 채택 신고 누적, 명확한 위반 정황, 피해 가능성이 큰 경우다.

## 제재 기준

피신고자의 채택 신고 누적 횟수가 3회이면 3일 정지를 검토한다.
4회이면 10일 정지, 5회이면 30일 정지, 6회 이상이면 영구 정지를 검토한다.
최종 처분은 관리자가 실제 신고 처리 API를 통해 결정한다.
