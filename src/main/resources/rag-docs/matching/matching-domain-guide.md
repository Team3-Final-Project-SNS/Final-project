# 매칭 AI 도메인 검색 가이드

## 의미 검색 대상

매칭 AI의 RAG 검색 대상은 게시글 한마디, 장소명, 메뉴 표현, 모집 조건, 후기와 매너 평가 맥락이다.
사용자가 정확한 키워드를 말하지 않아도 "조용한 밥", "많이 먹는 메뉴", "매너 좋은 사람" 같은 의미를 찾는 것이 목적이다.

## Tool Calling 결합

RAG는 의미적으로 관련 있는 후보를 찾는 역할을 한다.
최종 추천 전에 Tool Calling으로 모집 상태, 약속 시간, 모집 인원, 책임비, 사용자의 포인트 가능 여부를 MySQL 원본 데이터 기준으로 다시 확인한다.

## 멀티턴 대화 메모리

최근 N턴은 직접 프롬프트에 넣고, 오래된 대화는 벡터화해 현재 질문과 유사한 대화만 검색한다.
"아까 두 번째 추천이랑 비슷한 거" 같은 질문은 대화 메모리 검색으로 이전 추천 맥락을 복원한다.
